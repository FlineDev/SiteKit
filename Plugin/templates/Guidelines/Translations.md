# Translation Style Guide

This file documents per-language translation rules, terminology, formality preferences, and the author's voice in each target language.

> **How this file improves over time**: The `sitekit:i18n` skill reads this file before translating. After you edit an AI-translated draft, the skill compares the *first* AI draft against the final version (using the full session history of your corrections) and proposes additions to this file based on the patterns it finds. Over time, the file becomes a living reflection of your real translation decisions – not a one-time elicitation.

> **Default location**: `Guidelines/Translations.md` at the project root. To use a different path, set `localization.styleGuidePath` in `SiteConfig.yaml`.
