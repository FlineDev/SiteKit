# French (fr) – Web Content Style Guide

Read `principles.md` first. France-French (fr-FR) rules for SiteKit content. For Canadian French, see `fr-CA.md`.

## Address & formality
- **"vous"** is the safe default for a public website, even for a casual brand. **"tu"** suits a deliberately informal, personal, or young-audience site – decide once and keep it.
- French prose favours slightly more formal, complete sentences than English marketing copy; avoid clipped fragments where a full clause reads better.

## Quotation marks
- Use guillemets « … » **with a non-breaking space inside both marks**: `« comme ceci »`. Nested quotes use English-style “ … ”.

## Punctuation & spacing
- Put a **thin (non-breaking) space before** `:` `;` `!` `?` `%` and inside guillemets – e.g. `Bonjour !`, `50 %`. This is the strongest native-speaker signal.
- The colon before a list or quote is common and idiomatic.

## Numbers & dates
- Decimal **comma**, thousands **non-breaking space**: `1 234,50 €` (currency symbol after the amount, with a space). Never reformat numbers inside placeholders.
- Dates: `14 juin 2026` (day first, **month lowercase**, no comma).

## Capitalization & terminology
- **Sentence case** for headings and titles. Do **not** capitalize months, weekdays, languages, or nationalities as adjectives (`le français`, `lundi`, `juin`).
- Common web terms: `article`/`billet` (post), `s'abonner` (subscribe), `infolettre` or kept `newsletter`, `confidentialité` (privacy), `mentions légales` (imprint/legal notice).

## Common pitfalls
- "actuellement" means *currently*, not *actually* (`en fait`). "librairie" is a *bookshop*; a software library is `bibliothèque`.
- Avoid over-borrowing English (`email` → `courriel`/`e-mail`; `digital` → `numérique`).
- Elide and contract correctly (`l'article`, `d'un`, `qu'il`).
