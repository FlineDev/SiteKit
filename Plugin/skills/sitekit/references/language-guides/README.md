## A language not listed here?

[`principles.md`](principles.md) covers the categories that matter for any language. To add a dedicated guide, copy the structure of an existing file – address & formality, quotation marks, punctuation & spacing, numbers & dates, capitalization & terminology, common pitfalls – and fill in the target language's conventions for web prose.
